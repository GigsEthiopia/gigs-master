# Frontend

# Pre-requisite
    Install node and npm and pnpm 

## Usage

### To start the development server
npm run dev

### To serve the contents of the dist directory
npm run serve

### To build the project
npm run build

### To lint the TypeScript files
npm run lint

### To preview the project
npm run preview

## Clean (Not typically needed for Vite projects)
npm run clean

## Testing (Depends on your testing setup, may vary)
npm test


### To resynchronize `.gitignore`
    git rm -r --cached .
    git add .
    git commit -m "Resynchronize .gitignore"
